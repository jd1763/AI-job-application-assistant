Software Engineer I — Data Platform

Full-time · New York, NY (Hybrid) · Early-career

About the team
We build the data platform that powers analytics, ML, and product experimentation across the company. Our stack includes Python, Java, SQL, Airflow, Kafka, Spark, and Snowflake running in AWS.

What you’ll do

  • Build and maintain data ingestion pipelines (Kafka → S3 → Spark) with strong SLAs.
  • Develop internal services in Python or Java with high code quality and tests.
  • Author efficient SQL for analytics/metrics and optimize Snowflake queries.
  • Partner with product and DS to ship features behind A/B experiments.
  • Contribute to CI/CD and infrastructure as code on AWS (Docker, Terraform).

Basic qualifications

  • BS in CS (or equivalent) completed or in final year.
  • Experience with at least one: Python or Java.
  • Comfortable with SQL and data modeling.
  • Familiar with Linux, Git, and unit testing.

Preferred qualifications

  • Hands-on with Kafka, Spark, or Airflow.
  • Cloud experience with AWS (S3, IAM, ECS/EKS).
  • Nice-to-have: React, TypeScript, GraphQL for internal tools.

Benefits

  • Health, dental, 401(k) with match, generous PTO, learning stipend.
